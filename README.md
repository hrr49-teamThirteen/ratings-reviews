## Endpoints:
### Images:
#### /api/ratings/images/insert/:prodId/?loc=’path’
##### Insert an image
##### Params:
- prodId = product id
- loc = image path

#### /api/ratings/images/:prodId
##### Returns a list of images for a product id
##### Params:
- prodId = product id

#### /api/ratings/images/update/:imgId/:prodId?loc=’path’
##### Update an image
##### Params:
- imgId = image id
- prodId = product id
- loc = image path

#### /api/ratings/images/delete/:imgId
##### Delete an image
##### Params:
- imgId = image id
